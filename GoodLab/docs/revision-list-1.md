# 1차 수정 및 보완사항

1. 문서 작성(문서 편집) 페이지에서, 이미지를 넣을 수 있는 기능이 있어야 한다. (노션(Notion) 스타일의 이미지 넣기 기능과 같은 형식)

2. 문서 작성(문서 편집) 페이지에서, 문서 속에 문서를 넣을 수 있는 기능을 추가해야한다. 문서 속의 문서를 클릭하면 해당 문서의 내용이 나오는 형식이어야 한다. (노션(Notion) 스타일의, 페이지 속의 페이지 같은 형식)

3. 내 방에서 각각의 팀에서 사용되는 공지사항 기능을 추가해야한다. 각각의 팀이 개별적으로 사용하는 공지사항 기능이 있어야한다.

4. Notion 연동 기능은 필요없다. Notion 연동 기능은 모두 제거한다.

5. 내 방에서 각각의 팀에서, 팀원 목록에서 팀장 변경 시, 팀장은 가장 상단에 표시되어야 한다.
